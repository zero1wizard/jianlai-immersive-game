# 技术方案与实现计划

## 技术选型（首版）

- 语言：Node.js（快速迭代叙事原型）
- 架构：数据驱动事件系统（JSON 配置 + JS 结算器）
- 持久化：后续接入 SQLite（当前内存态）

## 模块划分

- `src/main.js`：游戏回合驱动、输入输出
- `src/systems.js`：角色状态、事件池、结算逻辑
- `config/world.json`：世界区域与初始事件配置

## 编码规范

- 规则与数据分离
- 每个事件可独立测试
- 保持纯函数优先（便于平衡性模拟）

## 后续演进

1. Web UI（React + Tailwind）
2. 存档系统（SQLite + Prisma）
3. 事件编辑器（可视化）
4. AI NPC 对话层（可插拔模型）
