# GDD：受《剑来》启发的沉浸式修真权谋游戏（完整方案）

## 1. 产品定位

- **类型**：叙事驱动 RPG + 关系策略 + 轻量经营
- **一句话**：在“修行与人心博弈”中，用选择塑造自己的道与天下格局。
- **目标体验**：
  - 角色成长的爽感（破境、悟道、得宝）
  - 权谋选择的压迫感（站队、背刺、取舍）
  - 叙事沉浸感（因果回响、角色记忆、江湖传闻）

## 2. 世界观拆解

- **结构层级**：
  1. 凡俗王朝（政局、商路、民心）
  2. 宗门与书院（修行体系、话语权）
  3. 天地异象（秘境、天劫、机缘）
- **冲突主轴**：
  - 人道秩序 vs 仙道超脱
  - 门派利益 vs 个人道心
  - 短期生存 vs 长期大道

## 3. 核心循环（Core Loop）

1. **游历**：在城镇/山门/秘境获取线索
2. **抉择**：面对事件作出价值判断（义、利、情、道）
3. **结算**：资源、关系、名望、道心变化
4. **成长**：突破境界、习得术法、改变势力版图
5. **反馈**：世界新闻与 NPC 态度回响到后续剧情

## 4. 关键系统设计

### 4.1 角色成长系统

- 属性：根骨、悟性、心境、气运
- 资源：灵石、丹药、功法熟练度
- 境界：凡骨→炼气→筑基→金丹→元婴（完整版）
- 机制：
  - 突破消耗资源与心境阈值
  - 心境不足时易走火入魔（负面状态）

### 4.2 关系与势力系统

- 势力：书院、仙府、江湖、王朝
- 变量：好感、信任、恐惧、债务（人情）
- 行为后果：
  - 站队获得短期收益，但锁死部分剧情
  - 中立可扩展信息面，但资源受限

### 4.3 叙事与事件引擎

- 事件由“标签条件 + 权重”触发
- 玩家选择会写入“因果账本”
- 同一事件在不同道心/名望状态下产生不同后续

### 4.4 经济与经营

- 主资源：灵石（货币）、材料（锻造）、情报（隐性资源）
- 中后期开放：
  - 洞府经营
  - 商路投资
  - 弟子培养

## 5. 叙事结构

- **卷式结构**：
  - 卷一：入世（认知世界与立场）
  - 卷二：破局（势力冲突与个人抉择）
  - 卷三：问道（大道选择与终局）
- **多结局**：王道、霸道、逍遥、殉道等

## 6. 商业化设计（若发行）

- 买断制本体 + 资料片（优先）
- 外观与配音包（不影响数值公平）
- UGC 剧情编辑器（社区长线）

## 7. 研发里程碑

- M1（2-4周）：文字原型可玩（核心循环 + 三章剧情）
- M2（4-8周）：Web 可视化 + 存档 + NPC关系网
- M3（8-12周）：数值平衡 + 多结局 + 首轮测试
- M4（12周+）：美术、音频、商业化准备

## 8. 风险与对策

- **叙事成本爆炸**：建立事件模板库 + 变量化文本
- **数值崩盘**：自动化模拟局 + 关键指标监控
- **内容同质化**：引入“因果账本”保障差异化回响

## 9. MVP 范围（本次代码实现）

- 可运行 CLI 原型
- 3 个区域、9+事件、基础成长与关系系统
- 单局 10 回合可完整体验“成长-抉择-反馈”闭环
