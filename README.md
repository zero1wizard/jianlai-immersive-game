## 运行

```bash
cd jianlai-immersive-game
node src/main.js
```

## 首版原型已实现

1. **核心循环**：游历（探索）→ 抉择（事件）→ 结算（资源/关系/境界变化）
2. **关键资源**：灵石、名望、心境、伤势
3. **成长系统**：凡骨→炼气→筑基（简化演示）
4. **势力关系**：书院、仙府、江湖帮派三方好感
5. **沉浸文本**：事件叙事 + 选项反馈
